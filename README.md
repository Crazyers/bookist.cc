# [BOOKIST.CC](http://bookist.cc) FRONT-END PROJECT

This is the front-ent project of [bookist.cc](https://bookist.cc) website.

## Quick Start

Make sure you've installed Node and Git.

1.  Clone this repository
2.  Run following comands

```bash
# install packages
npm install

# development
npm run dev

# production
npm run build
npm start
```

## Features

- Vue + Nuxt + Axios
